## Pre-requisites:
  
  php 8.2
  
  mongodb 1.15
  
  redis
  
  mysql
 
 
## Working:

To run it install composer and then run
`composer require mongodb/mongodb`

To run redis install redis and add to php.ini and
`redis-clis -h localhost`

To run mysql 
`sudo mysql -u root -p`

To run php
  `sudo php -S localhost:3000`

Note: always run php as root
